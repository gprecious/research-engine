# Intent Questions Fallback

Used when preview (Stage 2) fails and dynamic questions can't be generated.

Ask these 3 fixed questions:

1. **왜 이 자료를 분석하시나요?**
   - (a) 학습 / 이해
   - (b) 업무 적용 / 의사결정
   - (c) 공유용 요약
   - (d) 기타 (자유 서술)

2. **어떤 관점에 가장 집중해드릴까요?**
   - (a) 개념 · 이론
   - (b) 실행 · 구현 세부
   - (c) 장단점 · 트레이드오프
   - (d) 기타 (자유 서술)

3. **귀하의 배경지식 수준은?**
   - (a) 입문 — 용어부터 풀어 설명 필요
   - (b) 중급 — 요점 중심, 배경은 짧게
   - (c) 전문가 — 전문 용어 그대로, 새로움에 집중
